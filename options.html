Text Explainer Settings

This extension requires an OpenAI API key in order to function. No key is currently set.



Cost. You can generate an API key by creating an account at platform.openai.com/api-keys. OpenAI requires a credit card and a minimum $5 prepaid balance. You pay for what you use, but because this extension makes only lightweight requests: 100 uses will cost less than one cent.

Security. Your API key is stored locally in your browser using Chrome’s extension storage. It is never shared with anyone and is sent only to the OpenAI API when explanations are requested. To be extra safe, you can also specify a spending ceiling (e.g., $10) when setting up the API key. To add an extra layer of safety, you can set a usage limit or spending cap in your OpenAI account (for example, $10). This ensures the key can't generate unexpected charges beyond the limit you choose.
